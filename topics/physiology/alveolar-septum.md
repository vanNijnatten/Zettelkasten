# Alveolar Septum

lung tissue that separates adjecent alveoli

Consists of (**minimally**):
- basement membrane
  - type 1 pneumocytes
  - type 2 pneumocytes
  - capillary endothelium
Thicker alveolar septum **also** consists of:
- elastic fibers
- type 1 collagen
- interstitial cells
- smooth muscle cells
- mast cells
- lymphocytes
- monocytes
